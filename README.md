# SaaS Dashboard
## Project Overview
   This is a SaaS Dashboard built using HTML,Tailwind CSS and JavaScript.
   It demonstrates a modern dashboard interface with reusable components,responsive layout and interactive features

   ## Features
      Sidebar navigation(Home,Analytics,Reports & Settings)
      Header
      Dashboard Cards
      Modern UI using Tailwind
      Responsive Layout

   ## Technologies Used
       HTML5
       Tailwind CSS

  ## Design System
  ### Colors
     Primary - Blue
     Secondary - Gray
     Background - Light Gray
  
  ### Components
      Cards
      Sidebar
